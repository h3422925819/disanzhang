## 🎯 支持的图表类型

基于原Jupyter Notebook中的matplotlib图表，本网站实现了以下可视化类型：

### Chart.js支持的图表
- **折线图** (Line Plot) - 数据趋势展示
- **柱状图** (Bar Plot) - 分类数据对比
- **堆积柱状图** (Stacked Bar Plot) - 多系列数据叠加
- **水平柱状图** (Horizontal Bar Plot) - 横向数据对比
- **饼图** (Pie Plot) - 部分与整体关系
- **环形图** (Doughnut Plot) - 增强版饼图
- **散点图** (Scatter Plot) - 数据分布展示
- **气泡图** (Bubble Plot) - 三维数据可视化
- **雷达图** (Radar Plot) - 多维度数据对比
- **极坐标图** (Polar Area Plot) - 极坐标系数据

### D3.js支持的图表
- **箱线图** (Box Plot) - 数据分布统计
- **直方图** (Histogram) - 频率分布
- **误差棒图** (Error Bar Plot) - 数据不确定性展示

## 🚀 使用方法

### 快速开始
1. 直接在浏览器中打开 `index.html`
2. 选择想要创建的图表类型
3. 输入数据或点击"加载示例数据"
4. 实时查看生成的图表

### 第3章功能使用
1. **图例控制**
   - 勾选"显示图例"控制图例显示
   - 选择图例位置（顶部、底部、左侧、右侧）

2. **网格控制**
   - 勾选"显示网格"启用网格
   - 选择网格方向（X轴、Y轴、双向）
   - 选择网格样式（实线、虚线、点线）

3. **参考线和区域**
   - 勾选"显示参考线"添加参考线
   - 勾选"显示参考区域"添加参考区域
   - 勾选"显示注释"添加数据点注释
   - 勾选"显示表格"查看数据表格

4. **示例数据**
   - 选择"正弦余弦函数对比"查看3.1图例示例
   - 选择"电影票房排行榜"查看3.4网格示例
   - 选择"随机散点分布"查看3.5参考线示例

### 数据格式要求

#### JSON格式
```json
{
  "labels": ["标签1", "标签2", "标签3"],
  "datasets": [{
    "label": "数据系列1",
    "data": [10, 20, 30]
  }]
}
```

#### CSV格式
```csv
label,value1,value2
类别A,10,15
类别B,20,25
类别C,30,35
```
